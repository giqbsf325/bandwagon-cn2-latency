## 先说清楚：搬瓦工有几种 CN2？

搬瓦工 CN2 不是一种线路，是三种，延迟和价格差距挺大的，买之前搞清楚这个很重要。

**CN2 GT（普通版）**：全称 Global Transit，AS4809 编号，去程走电信 CN2，但回程不一定，高峰期有拥堵。这是最便宜套餐（KVM/CN2 系列）能用到的线路，目前在 DC3、DC8 机房，适合预算紧张、对延迟没那么敏感的用户。

**CN2 GIA-E（主力版）**：GIA 全称 Global Internet Access，三网回程全走 CN2 GIA（AS4809 高端段），DC6、DC9 双机房都是这个线路，另外还可以切到日本软银（JPOS\_1）和荷兰9929（EUNL\_9）。晚高峰抗压能力比 GT 强多了，这是目前搬瓦工性价比最高的线路组合。

**CN2 GIA（香港/日本高端版）**：物理距离近，香港机房对国内延迟能做到两位数。这是专属线路，不能从 CN2 GIA-E 套餐迁移过来，必须单独买香港套餐。

## 为什么 CN2 GT 和 CN2 GIA 延迟差这么多

说实话不完全是延迟的问题，更多是**稳定性**。

CN2 GT 的延迟在白天其实也不差，和 GIA 差距有限。但到了晚上 8 点到 11 点，普通线路就开始堵，丢包率可能到 30% 以上。这时候你会发现页面加载变慢、SSH 连上断线、视频卡顿。

CN2 GIA 用的是电信最贵的优先级通道，容量非常有限（搬瓦工官方的说法是每 Mbps 价格高达 \$120，资源稀缺），好处就是这条路上的流量少，拥堵程度几乎感受不到。

所以两者的核心区别不是延迟数字，是**晚高峰时段的稳定性**。如果你主要在工作时间用，CN2 GT 也够；如果晚上经常用，GIA 的差距就体现出来了。
